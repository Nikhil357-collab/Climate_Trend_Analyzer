# 🌍 Climate Trend Analyzer

## 📌 Overview
A data science project that analyzes historical climate data to identify trends, detect anomalies, and forecast future climate patterns.

## 🎯 Objectives
- Analyze temperature, rainfall, and humidity trends
- Detect anomalies (heatwaves, extreme rainfall)
- Perform time-series forecasting
- Visualize insights using interactive dashboards

## 📊 Dataset
- Synthetic dataset (realistic simulation)
- Daily data from 2018–2023
- Features:
  - Date
  - Temperature (°C)
  - Rainfall (mm)

## ⚙️ Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- Scikit-learn, Statsmodels (ARIMA)
- Streamlit

## 🔄 Workflow
1. Data Generation / Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Trend Analysis
5. Anomaly Detection
6. Forecasting
7. Visualization Dashboard

## 📈 Results
- Identified increasing temperature trends
- Detected extreme weather anomalies
- Forecasted future climate patterns
